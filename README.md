# 📊 Sales Dashboard (Excel Project)

## 📌 Objective
To analyze sales performance using Excel and create an interactive dashboard to track key business KPIs.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting

## 📂 Dataset
The dataset includes:
- Sales representatives (Name)
- Total Calls
- Calls Reached
- Deals Closed
- Deal Value ($)
- Monthly performance data

## 🔍 Key Features
- KPI cards (Total Calls, Calls, Deals, Revenue)
- Sales performance by individual
- Monthly deals analysis
- Regional service level visualization (map)
- Interactive filtering by salesperson

## 📊 Insights
- Identified top-performing sales representatives
- Tracked monthly deal trends
- Analyzed conversion from calls to deals
- Compared regional performance levels
